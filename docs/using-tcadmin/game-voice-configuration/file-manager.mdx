# File Manager

- `Editable Extensions` - Enter the extensions that can be edited with the text editor. You can specify more than one extension by separating with `';'`. For example `*.log;*.txt`
- `Max Editable Size` - The maximum file size in KB that can be edited. Set to 0 to disable editing.
- `Default File Name` - The default file name shown when the user clicks on the 'New File' button.
- `Enable downloads` - Enable file downloads.
- `Enable uploads` - Enable file uploads.
