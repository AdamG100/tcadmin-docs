# Punkbuster Settings

- `Enable Punkbuster update` - Enable the punkbuster update for this game.
- `Relative PB Path` - The relative path to the game server's Punkbuster folder.
- `PB Download Url` - The url to the game's download page. TCAdmin will download all files availabel for download. To find the url for a specific game visit [http://www.punkbuster.com](http://www.punkbuster.com).

This value can also be a link to `pbsvnew.dll` but some game servers require more files.
