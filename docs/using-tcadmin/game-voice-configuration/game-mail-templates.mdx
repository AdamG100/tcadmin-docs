# Game Mail Templates

If you want to have a different mail template first you must create it in `System` (on the left-hand navigation) > `General Settings` > `Mail Server Templates`.

- Install User Template - Select the template sent to the user when the service is created.
- Install Admin Template - Select the template sent to administrators when the service is created.
- Reinstall User Template - Select the template sent to the user when the service is reinstalled.
- Reinstall Admin Template - Select the template sent to administrators when the service is reinstalled.
