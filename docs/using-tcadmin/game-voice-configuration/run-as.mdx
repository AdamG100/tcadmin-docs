# Run As

 - Run As
   - Select the Windows user that will be used to run the process.
   - `LocalSystem` : This user has full permissions on the server and can be a security risk.
   - `TCAGame` : This user is a guest created by TCAdmin with permissions limited to the game server directory.
   - `Specific User` : Lets you run the process as a Windows or Linux account that you have preconfigured. The user is not created if it does not exist.
