# Windows Installation

1. Make sure you have installed the [System Requirements](/getting-started/system-requirements).
2. Download the [TCAdmin installer](https://downloads.tcadmin.net/installer/tcadmin2setup.exe)
3. Execute and install. Change the installation directory as needed.
4. Follow the [Initial Configuration](/getting-started/initial-configuration) guide to configure TCAdmin.
