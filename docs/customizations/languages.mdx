# Languages

## Translations

Translations are divided into 3 parts:

1. **Web Forms**
   Translate controls and messages from web pages.

2. **Page Icons & Columns**
   Translate page titles, icons, and grid columns.

3. **Other Components**
   Translate components that are shared among the web pages and server monitors. This section also includes MVC content.

## Importing
To import a language file, follow these steps:
1. Go to `System` > `General Settings` > `Languages`.
2. Click on the **Import** button.

## Exporting
To export a language file, follow these steps:
1. Go to `System` > `General Settings` > `Languages`.
2. Select a language.
3. Click on the **Export** button.
